# Rebar Optimizer Pro

**Rebar Optimizer Pro** هو برنامج لحساب وتحسين استخدام أسياخ الحديد في مشاريع البناء، مع توليد تقرير PDF جاهز للطباعة مع:

- MainBar: الأسياخ المطلوبة حسب القطر والطول والعدد والوزن.
- Waste Bars: المخلفات لكل قطر وطول.
- Purchase 12m Bars: عدد القضبان اللازمة لكل قطر، الوزن الإجمالي، والتكلفة.
- Cutting Instructions: توزيع القطع على قضبان 12 متر مع أنماط القطع.
- توليد تقرير PDF يشمل شعار **NovaStruct Company**.
